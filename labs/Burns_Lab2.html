Part A Answer:
As we sum for more days, the probability mass function width increases, and moves the peak of the function to the right. Part B Answer: The second plot above shows the pmf for the Poisson distribution after 5 days. We can see mathematically that it remains a Poisson distribution because the only manipulation done to it was convoluting the original distribution with itself 5 times, which essentially shifts the peak. The original peak was at 2, the shifted peak after 5 days should be 2*5=10, and the second chart confirms this. However, it still looks Poissonian because by summing over 5 days, that isn't very many days, and so the peak hasn't shifted significantly. If we were to sum over a larger amount of days, we would see the distribution approach a Gaussian shape.

Part C Answer:
As you average over more and more days, the width of the distribution decreases because when you average all the distributions from many days, the probability of getting measurements far from the mean decreases. The peak, however, remains in the same place as the initial distribution. When comparing the average distribution to the sum distribution we see the average distribution converges to a Gaussian distribution faster than the sum. It takes roughly 500 days for the average to converge, and roughly 1200 for the sum to converge. When we compare this to the Central Limit Theorem, both cases reaffirm the CLT because as you increase the number of days, both the sum and average distributions move toward a Gaussian distribution, but the average does this faster than the sum for this particular Poisson distribution.

Part C Answer:
If you have a trials factor of 10000, the signal would have to be 1.31 times brighter in order to be considered a significant discovery.
The sensitivity penalty for a trials factor of 10k is so low in these particular distributions because the probability of the background producing a signal greater than 1 was already so low in the initial distribution, so when the number of trials was increased, it did not have a significant effect on the new probability distribution. We can see the probability of seeing a signal of, say, 5 in the initial distribution was around 10e-9, so the probability of seeing that signal in the distribution after 10000 trials was 10e-5. Due to this, we can see the change in probability is incredible tiny when comparing 1 trial to 10000 trials, and so the sensitivity penalty of the trial factor is not very severe for these distributions.

Part D Answer:
As you increase the magnitude of the trials factor by 10, the 5 sigma threshold it produces increases by between .3-.4 signal strength each time. So for example, if we increase the trials factor from 10,000 to 100,000, the 5-sigma signal value goes from 6.55 to 6.89. So the signal needs to increase in strength by roughly .3 to .4 for each increase in trial factor by a factor of 10.
